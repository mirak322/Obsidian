2) ==Какой тип данных передается по ссыкле?==

**Скалярные типы (int, float, string, bool)** — копируются по значению, но могут быть переданы по ссылке через `&`.

**Составные типы (array, object)** — объекты **по умолчанию передаются по ссылке**, массивы — копируются, но можно использовать `&`.

4) ==PHP интерпретируемый или компилируемый? Почему?==

**PHP — интерпретируемый язык.**

-Код **не компилируется в машинный код заранее**.
-PHP-скрипт **читается и выполняется интерпретатором (php-fpm, CLI) построчно**.
-Причина: PHP предназначен для **динамической генерации веб-страниц**, где важна гибкость, а не скорость компиляции.
