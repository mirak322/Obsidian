cntr + /  на всю строку - комментируем код

Вывод массива:

echo $person; - не раб так как массив

другая функция:

print_r($person);

У массива есть индексы, они начинаются с 0 и так далее

например
print_r($person[0]); - будет выведенно имя Victor

Мы можем сами задавать индексы например

$person = ['name' => 'victor']

на выводе будет [name] => Victor
[0] => 20
и тд

Ключ и значение

Массив рекомендуется записывать по строчкам

print_r($person['name']); - Victor

cntr + shift + enter = закрывает всю строчку ;
