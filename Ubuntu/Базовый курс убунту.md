bin (binaries) - файлы утилит (cat,ls и др)

dev (devices) - файлы устройств (клава принтер)
etc etcetera - файлы конфигов программ
home - файлы пользователя
lib  - файлы библеотек
media - точкимонтирования для сменных носителей
mnt (mount) - используется для монтирования временных файлов
opt (optional) - файлы допольнит программ
proc (processes) файлы запущ процессов
root - домашнаяя директория пользователя root
sbin (system binaries) - файлы для администрирование системы(резервное копирование и тд)
srv (service) - данные для окруж системы
tmp (temporary) -временные файлы
usr (user) - пользовательские приложения и утилиты
var (variables) - файлы для работы программ
