**git clone - закачать к себе проект**
**git add - добавить изменения ;автоматом определяются все измененные файлы**
**git commit - зафиксировать изменения**
**git push - отправить изменения**
**git pull - скачать проект с изменениями**
**merge - соединить, слить воедино (объединить изменения)**

Если  у нас есть проект и мы хотим создать аналог проекта то
создаем ветку

**git branch - создать аналог на основе исходного проекта** 
**git checkout - дает возможность переключаться между ветками**

Есть Master ветка - оригинал
Branch - создание доп ветки
